git remote add origin git@gitee.com:liu-yu-mitumao/ceshi.git

git push --set-upstream origin master的原因以及解决方法
这是因为本地的分支没有和远程分支建立联系，需要执行以下代码就可以正常push了：
git push --set-upstream origin master
